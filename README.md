## 📝 Описание проекта

**Random Task Generator** — это графическое приложение на Python с использованием библиотеки Tkinter, которое помогает генерировать случайные задачи для продуктивного дня. Приложение сохраняет историю сгенерированных задач, позволяет добавлять собственные задачи и фильтровать их по категориям.

Проект разработан в рамках **итоговой аттестации** по курсу **"Python с нуля: первые шаги в программировании"**.

### Основные возможности:

| Функция | Описание |
|---------|----------|
| 🎲 Генерация случайных задач | Выбор случайной задачи из предопределённого списка |
| ➕ Добавление своих задач | Возможность пополнить список задач с указанием категории |
| 📜 История генераций | Автоматическое сохранение всех сгенерированных задач с временной меткой |
| 🔍 Фильтрация по категориям | Просмотр истории только по выбранной категории (учёба/спорт/работа) |
| 💾 Сохранение в JSON | Данные автоматически сохраняются при каждом изменении |
| 🚫 Валидация ввода | Проверка на пустые строки и минимальную длину названия |
| 🔄 Загрузка при запуске | Приложение восстанавливает историю и добавленные задачи |
